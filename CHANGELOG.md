# Changelog

All notable changes to this project will be documented in this file.

## [0.2.0] - 2026-02-26

### Changed
- Release `0.2.0`.
- Major update to the project structure and behavior.
- Dependency updates and maintenance improvements.

### Fixed
- Linux fullscreen fixes.
- macOS multi-monitor and monitor management fixes.
- Antialiasing and blank-screen handling fixes.

## [0.1.0] - 2025-01-18

### Summary
- Introduced major app capabilities (IPC, annotations, manual SDP, interactive save path, popup/UI improvements).
- Improved capture and streaming stability with multiple bugfixes and cleanup passes.
- Extended cross-platform behavior with fixes for macOS, Linux fullscreen, monitor handling, and encoding reliability.

## [0.0.0] - 2024-07-30

### Added
- Initial project setup and first working foundation of Castify.
